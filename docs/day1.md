## 💡 关键理解

### 1. 线程到数据的映射
```cpp
const uint x = blockIdx.x * blockDim.x + threadIdx.x;  // C的行索引
const uint y = blockIdx.y * blockDim.y + threadIdx.y;  // C的列索引
```
- 每个线程负责计算C的一个元素C[x][y]

### 2. 索引计算公式（行主序存储）
```cpp
A[x * K + i]  // A矩阵第x行第i列：跳过x行(x*K) + 第i个元素
B[i * N + y]  // B矩阵第i行第y列：跳过i行(i*N) + 第y个元素
C[x * N + y]  // C矩阵第x行第y列
```

**具体例子** (M=3, K=2, N=4)：
- A[1][1] = A[1*2 + 1] = A[3] ✅
- B[1][2] = B[1*4 + 2] = B[6] ✅
- C[1][2] = C[1*4 + 2] = C[6] ✅

### 3. 计算过程
对于C[x][y]的计算：
```cpp
for (int i = 0; i < K; ++i) {
    tmp += A[x * K + i] * B[i * N + y];
}
// 这是在计算A的第x行和B的第y列的点积
```
